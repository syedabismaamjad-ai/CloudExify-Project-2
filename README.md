#  Student Grade Management System

## Project Overview

**Student Grade Management System** is a Python-based application developed to help educational institutions and teachers manage student academic records efficiently. The system allows users to add student information, record grades, calculate averages, generate reports, and track student performance. It provides both a **Console-Based Interface** and a **Graphical User Interface (GUI)** built with Tkinter for an enhanced user experience.

This project was developed to demonstrate Python programming concepts including file handling, data management, user interfaces, and academic record processing.

***

## Objectives

* Track and manage student academic records efficiently.
* Record grades for multiple subjects.
* Calculate student averages automatically.
* Generate performance reports and rankings.
* Provide an easy-to-use console and graphical interface.

***

## Features

### Student Management

* Add new student records.
* Generate unique student IDs automatically.
* Search students by name.
* Delete student records.

### Grade Management

* Enter marks for different subjects.
* Edit existing student grades.
* Validate marks between 0 and 100.
* Calculate average marks automatically.

### Subjects Included

The system manages grades for the following subjects:

* Mathematics
* Physics
* English
* Computer
* Urdu

### Performance Reports

* Calculate student averages.
* Display PASS/FAIL status automatically.
* Generate class rankings.
* Show highest, lowest, and overall class averages.

***

## Data Storage

* Student records are stored in a CSV file (`students.csv`).
* Data can be saved and loaded automatically.
* Records remain available across multiple sessions.
